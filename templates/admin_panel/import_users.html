Instruções
Formato do Arquivo JSON:
[
  {
    "email": "usuario@exemplo.com",
    "first_name": "Nome",
    "last_name": "Sobrenome",
    "password": "senha123",
    "user_type": "client",
    "cidade": "São Paulo",
    "estado": "SP",
    "cep": "01234-567"
  }
]
Importante: O arquivo deve estar no formato JSON válido e conter todos os campos obrigatórios.
Tempo: A importação pode levar alguns minutos dependendo da quantidade de usuários.
Campos Obrigatórios
  • email: E-mail único do usuário
  • first_name: Primeiro nome
  • last_name: Sobrenome
  • password: Senha do usuário
  • user_type: "client" ou "supplier"
Campos Opcionais:
  • • cidade, estado, cep
  • • telefone, whatsapp
  • • descricao, especialidades
